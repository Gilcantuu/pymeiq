# Contributing to PymeIQ

Academic project for AI-101: Free-Stack Agentic Builder Studio.

## Conventions
- All UI, code, and commit messages in English.
- Commit style: type(scope): summary.
- Never commit .env.local.

## Local workflow
1. Clone the repo.
2. Run npm install.
3. Copy .env.example to .env.local and fill in Supabase values.
4. Run npm run dev and open http://localhost:3000.
